## ğŸ¨ Customization Examples

### Change Colors
Edit the CSS variables in the `<style>` section:
```css
:root {
    --color-primary: #FF5733;  /* Change primary color */
    --color-accent-1: #3498DB; /* Change accent */
}
```

### Add Your Logo
```html
<!-- Add before </head> -->
<img src="your-logo.png" style="position: fixed; top: 20px; right: 20px; max-height: 50px; z-index: 999;">
```

### Modify Footer
```html
<div class="footer">
    <span class="footer-left">Your Company</span>
    <span class="footer-center">Your Event</span>
    <span class="footer-right">Date | <span class="page-number"></span></span>
</div>
```
