What Is It?

A Claude Code skill that automatically generates professional presentations in multiple formats:

  • reveal.js HTML - Interactive web presentations
  • Marp - Simple markdown slides
  • Slidev - Vue-powered presentations
Best part: Just describe what you want, and Claude generates the complete presentation!
